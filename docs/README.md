구현 기능 목록
-----

* 자동차는 전진 또는 멈춤
* 자동차는 이름 가진다
    * 5자 이하만 가능
* 자동차 출력 시 이름을 같이 출력한다
* 사용자 입력
    * 자동차 이름(구분자 쉼표)
    * 몇 번의 이동
    * 잘못입력한 경우 IllegalArgumentException 발생
        * [ERROR] 로 시작하는 메시지 출력 후 입력 다시 받는다
* 전진 조건
    * 0~9 사이의 랜덤값 구한 후 4이상일 경우 전진, 3이하 멈춤
* 최종 출력
    * 누가 우승했는지
    * 우승자 복수이면 쉼표로 이름 구분
